# E-commerce Frontend - Módulo 2

Este es el proyecto de una tienda en línea de repuestos para celulares y vehículos, cumpliendo con los requisitos del MVP.

## Características:
- **HTML5 Semántico**: Uso de header, nav, main, section y footer.
- **Bootstrap 5**: Diseño responsivo (Mobile-first).
- **JavaScript**: Lógica de contador de carrito en tiempo real.
- **Git**: Control de versiones con commits descriptivos.
